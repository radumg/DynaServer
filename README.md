# DynaServer
A Dynamo Extension that allows bi-directional communication and control of Dynamo as a REST API.

<img src="https://github.com/radumg/DynaServer/blob/master/docs/DynaServerDemo.gif?raw=true" width="100%"/>


## Alpha notice
Please note the package is current in `alpha` version `0.1`, meaning it's not suitable for production use or deployment to your entire practice. Use at your own risk until a stable `1.0` version is achieved.
